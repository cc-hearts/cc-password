# cc-password

cc-password 是一款基于 Electron 和 Vue 3 的桌面端密码管理工具。

## 功能特点

- [x] 安全存储和管理密码
- [x] 支持创建、编辑和删除密码条目
- [ ] 自动生成强密码
- [ ] 密码强度评估
- [x] 支持多用户和多账户
- [x] 加密存储和数据同步
- [x] 快捷键操作

## 技术栈

- `Electron` - 用于构建跨平台桌面应用程序的开源框架
- `Vue 3` - 用于构建用户界面的渐进式 JavaScript 框架
- `Vue Router` - 用于管理应用程序路由的官方 Vue.js 路由器
- 其他依赖项详见 `package.json` 文件
